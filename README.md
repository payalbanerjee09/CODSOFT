# CodSoft Web Development Internship (Level 1)

Welcome to my **CodSoft Internship** repository! This repository contains all the tasks assigned during my Level 1 Web Development Internship. Each task has been crafted using clean code practices to ensure responsiveness and a modern UI/UX design.

---

🚀 Projects Overview

Here is a brief description of the three tasks completed during this internship:

 🎯 Task 1: Personal Portfolio Website
A clean, minimal, and fully responsive personal portfolio website designed to showcase my skills and contact details.
* **Tech Stack:** HTML5, CSS3
* **Key Features:** * Modern "Avatar Box" design (custom CSS icon instead of a profile picture)
  * Dynamic grid-based layout for skills and projects
  * Fully responsive design for mobile, tablet, and desktop screens

 🌐 Task 2: Landing Page
A professional landing page designed for a premium **Coffee Shop** brand, optimized for conversions and user engagement.
* **Tech Stack:** HTML5, CSS3
* **Key Features:**
  * Clean navigation bar with a hero section
  * Beautiful feature/product cards with elegant CSS hover effects
  * Structured pricing blocks and a strong Call-to-Action (CTA) button

### 🔢 Task 3: Calculator
A fully functional, modern dark-themed calculator capable of performing all basic mathematical operations.
* **Tech Stack:** HTML5, CSS3, JavaScript (ES6)
* **Key Features:**
  * Sleek dark-mode aesthetic with glow/hover effects on buttons
  * Real-time input handling using vanilla JavaScript
  * Handled Edge cases (like division by zero or consecutive operators) smoothly
